# Cloudlinux

Each task is done on a real machine.

## Task 1

Compile module </br>
`make`

Add module </br>
`sudo insmod hello_kernelcare.ko`

Show message module </br>
`dmesg | tail -n 5`

Delete module </br>
`sudo rmmod hello_kernelcare.ko`
